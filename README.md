Mapbox GL Example Styles
================
See https://www.mapbox.com/mapbox-gl-styles/ for live examples.


### Usage

```js
var mapboxGLMaps = require('mapbox-gl-styles');

// Get a style
mapboxGLMaps.styles['dark-v8'];

// Get sprites
mapboxGLMaps.sprites['dark-v8'];
```
